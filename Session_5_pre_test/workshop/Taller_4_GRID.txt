Taller Creativo de Diseño Web: Creando la Web de "Café Amanecer"

- Introducción:

Bienvenido al taller creativo donde transformaremos tus ideas en una página web única para "Café Amanecer", un negocio emergente que busca capturar la esencia del primer café del día. Este taller te guiará a través del proceso de diseño, desde la elección de la paleta de colores hasta la implementación de un diseño responsivo utilizando position, flexbox, y grid.

- Objetivo:

Crear un diseño web interactivo y atractivo para "Café Amanecer" que refleje calidez, energía y la frescura de un nuevo día. Utilizaremos tecnologías modernas de CSS para lograr un sitio web responsivo y visualmente atractivo.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Fase 1: Definición del Concepto

- Conceptualización:

Historia de la Marca: "Café Amanecer" simboliza un nuevo comienzo, ofreciendo una experiencia rejuvenecedora a sus clientes a través de su café gourmet. Se deleitan en gozar del mas alto reconocimiento de calidad y profesionalismo, pues cuentan con 3 de los mejores baristas del mundo.

Misión: Enriquecer cada mañana con el aroma y sabor de nuestro café, garantizando un despertar lleno de energía y positividad.
Elección de Paleta de Colores y Fuente:

Paleta de Colores: El cliente elegirá entre tres paletas prediseñadas que evocan calidez, energía y frescura. Debes elegir de entre 3 paletas de colores y definir los valores hexadecimal o rgb o rgba en la pseudo-clase root.

Fuentes de Google: Seleccionaremos dos fuentes de Google que complementen la estética y el mensaje de la marca. Puedes elegir cualquier fuente de google que considere.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Fase 2: Desarrollo del Diseño

Estructura con HTML y CSS:

- Crearemos la estructura básica de una página utilizando HTML5, incluyendo las secciones descritas en la seccion final "Entregables"..

Logo Creativo con HTML y CSS:

- Utilizaremos un <h1> para el nombre "Café Amanecer", aplicando un <span> a las últimas dos letras "er" para cambiarles el color y crear un efecto visual que sirva como logo.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Entregables:

Una página web completa para "Café Amanecer" que incluya:

	Un logo creativo hecho con HTML y CSS.
